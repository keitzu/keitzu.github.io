实现 NumArray 类：

  • NumArray(int[] nums) 使用数组 nums 初始化对象
  • int sumRange(int i, int j) 返回数组 nums 从索引 i 到 j（i ≤ j）范围内元素的总和，包含 i、j两点（也就是 sum(nums[i], nums[i + 1], ... , nums[j])）

示例：

提示：

  • 0 <= nums.length <= 104
  • -105 <= nums[i] <= 105
  • 0 <= i <= j < nums.length
  • 最多调用 104 次 sumRange 方法
题解

方法一：直接遍历i到j计算总和

方法二

初始化时先计算每个元素到0累计总和。需要计算i到j的总和只需要0到j的总和减去0到i - 1总和。
