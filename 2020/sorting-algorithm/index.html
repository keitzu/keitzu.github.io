排序算法是最基础的算法之一，可以使用的场景也很多。不同的排序算法有不同的优缺点，这里列举了八种常用的排序算法。

各个算法的复杂度和稳定性：

排序算法 平均时间复杂度 空间复杂度 稳定性
冒泡排序 $O(n^2)$ $O(1)$ 稳定
选择排序 $O(n^2)$ $O(1)$ 不稳定
插入排序 $O(n^2)$ $O(1)$ 稳定
希尔排序 $O(n^{\frac{3}{2}})$ $O(1)$ 不稳定
归并排序 $O(nlog_{2}n)$ $O(n)$ 稳定
快速排序 $O(nlog_{2}n)$ $O(log_{2}n)$ 不稳定
堆排序 $O(nlog_{2}n)$ $O(1)$ 不稳定
基数排序 $O(n*k)$ $O(n+k)$ 稳定

本页面所有代码均使用C实现，使用Leetcode的数组排序进行测试（题目：给你一个整数数组 nums，请你将该数组升序排列）,部分不够高效的排序算法在进行大量数据排序时会超出时间限制。部分排序算法参考排序数组-题解

1、冒泡排序

1.1、步骤

  1. 从头开始比较相邻元素，如果大小顺序错误则交换位置
  2. 往后移动继续比较，直到末尾（这时最大的元素应在末尾）
  3. 如果某一趟排序未进行交换操作则排序结束
  4. 忽略末尾已排序的元素，重复进行1与2直到排序完成

1.2、实现

int* sortArray(int* nums, int numsSize, int* returnSize){
    *returnSize = numsSize;
    int i, j;
    bool isFinished = true;
    for(i = 0; i < numsSize - 1; i++){
        for(j = 0; j < numsSize - i - 1; j++){
            if(nums[j] > nums[j + 1]){
                int temp = nums[j];
                nums[j] = nums[j + 1];
                nums[j + 1] = temp;
                isFinished = false;
            }
        }
        if(isFinished) break;
    }
    return nums;
}

2、选择排序

2.1、步骤

  1. 遍历数组找出其中的最小元素
  2. 将最小元素放置到已排序部分队尾
  3. 剩余元素重复1于2直至排序完成

2.2、实现

int* sortArray(int* nums, int numsSize, int* returnSize){
    *returnSize = numsSize;
    int i, j, min;
    for(i = 0; i < numsSize; i++){
        min = i;
        for(j = i; j < numsSize; j++){
            if(nums[j] < nums[min]){
                min = j;
            }
        }
        int temp = nums[i];
        nums[i] = nums[min];
        nums[min] = temp;
    }
    return nums;
}

3、插入排序

3.1、步骤

在要排序的一组数中，假定前n-1个数已经是有序序列

  1. 找到第n个数插入有序序列的位置k
  2. 将k到n-1位的数后移一位，第k位的值赋值位第n位的值
  3. 重复1与2直到排序完成

3.2、实现

int* sortArray(int* nums, int numsSize, int* returnSize){
    *returnSize = numsSize;
    int i, j;
    for(i = 1; i < numsSize; i++){
        int temp = nums[i];
        j = i - 1;
        while( j >= 0 && temp < nums[j]){
            nums[j + 1] = nums[j];
            j--;
        }
        nums[j + 1] = temp;
    }
    return nums;
}

4、希尔排序

4.1、步骤

  1. 取一个小于n的步长$d_1$，将带待排序元素分为$d_1$组
  2. 对每一组分别进行插入排序
  3. 取第二个步长$d_2$($d_1$>$d_2$),重复1与2直至排序完成

4.2、实现

int* sortArray(int* nums, int numsSize, int* returnSize){
    *returnSize = numsSize;
    int gap, i, j;
    int temp;
    for (gap = numsSize / 2; gap > 0; gap /= 2)
        for (i = gap; i < numsSize; i++) {
            temp = nums[i];
            for (j = i - gap; j >= 0 && nums[j] > temp; j -= gap)
                nums[j + gap] = nums[j];
            nums[j + gap] = temp;
        }
    return nums;
}

5、2路-归并排序

5.1、步骤

假定待排序表中n个元素为n个有序的子表，归并是指将两个有序表合成为一个新的有序表

  1. 将子表两两归并，得到n/2个长度为2或1的有序子表
  2. 重复1直至排序完成

5.2、实现

void Merge(int* nums, int startIndex, int midIndex, int endIndex, int* sorted)
{
    int left = startIndex;
    int right = midIndex + 1;
    int sortedIndex = 0;
    while (left <= midIndex && right <= endIndex) {
        if (nums[left] < nums[right]) {
            sorted[sortedIndex++] = nums[left++];
        } else {
            sorted[sortedIndex++] = nums[right++];
        }
    }

    while (left <= midIndex) {
        sorted[sortedIndex++] = nums[left++];
    }

    while (right <= endIndex) {
        sorted[sortedIndex++] = nums[right++];
    }
    sortedIndex = 0;
    while (startIndex <= endIndex) {
        nums[startIndex++] = sorted[sortedIndex++];
    }
    return;
}

void MergeSort(int* nums, int startIndex, int endIndex, int* sorted)
{
    if (endIndex <= startIndex) {
        return;
    }
    int midIndex = (startIndex + endIndex) / 2;

    MergeSort(nums, startIndex, midIndex, sorted);
    MergeSort(nums, midIndex + 1, endIndex, sorted);
    
    Merge(nums, startIndex, midIndex, endIndex, sorted);
    return;
}

int* sortArray(int* nums, int numsSize, int* returnSize)
{
    int* sorted = malloc(sizeof(int) * numsSize);
    MergeSort(nums, 0, numsSize - 1, sorted);
    free(sorted);
    *returnSize = numsSize;
    return nums;
}

6、快速排序

6.1、步骤

快速排序是对冒泡排序的一种改进算法。

  1. 在待排序表中任取一个元素p作为基准
  2. 通过一趟排序将待排序表分为两个子表，其中一个子表所有元素小于p，另一个子表多有元素大于p
  3. 递归地对上面两个字表进行1与2直至排序完成

6.2、实现

void quicksort(int* Nums, int left, int right){
    if(left<right){
        int i=left,j=right,temp=Nums[left];
        while(i!=j){
            while(i<j&&Nums[j]>temp)
                j--;
        if(i<j)
            Nums[i++]=Nums[j];
        while(i<j&&Nums[i]<temp)
            i++;
        if(i<j)
            Nums[j--]=Nums[i];
        }
        Nums[i]=temp;//
        quicksort(Nums,left,i-1);
        quicksort(Nums,i+1,right);

    }
}
int* sortArray(int* nums, int numsSize, int* returnSize){
    *returnSize=numsSize;
    quicksort(nums,0,numsSize-1);
    return nums;
}

7、堆排序

7.1、步骤

在排序的过程中，将待排序表看成是一棵完全二叉树的顺序存储结构，利用玩完全二叉树中双亲结点和孩子结点之间的内在关系，在当前无序区中选择关键字最大（最小）的元素。

  1. 用待排序表构建初始堆
  2. 输出堆顶元素
  3. 用剩余元素构建大顶堆（小顶堆），重复1与2直至排序完成

7.2、实现

void AdjustDown(int* nums,int a,int numsSize){   //向下调整
    int temp = nums[a];
    for(int i=2*a+1;i<numsSize;i=2*i+1){
        if(i+1<numsSize && nums[i+1]>nums[i]){
            i++;
        }
        if(temp >= nums[i]){
            break;
        }
        nums[a] = nums[i];
        a = i;
    }
    nums[a] = temp;
}
int* sortArray(int* nums, int numsSize, int* returnSize){
    (*returnSize) = numsSize;
    if(numsSize < 1){
        return nums;
    }
    for(int i=numsSize/2;i>=0;i--){          //建大根堆
        AdjustDown(nums,i,numsSize);
    }
    for(int i=numsSize-1;i>0;i--){       //排序
        int temp = nums[0];
        nums[0] = nums[i];
        nums[i] = temp;
        AdjustDown(nums,0,i);
    }
    return nums;
}

8、基数排序

基数排序是一种特别的排序方法，它采用多关键字排序思想，借助“分配”和“收集”两种操作对单逻辑关键字进行排序。基数排序又分为最高位优先排序和最低位优先排序。
